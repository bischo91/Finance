## Getting Started

### Prerequisites

The program requires [Python](https://www.python.org/), and the following Python packages:
* PyQt5 (5.15.4)
* Datetime (4.3)
* numpy (1.20.2)
* matplotlib (3.4.1)
* csv (1.0)
* openpyxl (3.0.7)
* xlsxwriter (3.0.2)
